Help Desk Hexagonal 3.0 Sistema de Gerenciamento de Chamados

Bem-vindo ao nosso sistema Help Desk V3, desenvolvido com arquitetura hexagonal!
Este projeto demonstra a aplicação prática da arquitetura hexagonal (ou ports and adapters) em um sistema de gerenciamento de chamados. Utilizando Angular no frontend e Spring Boot no backend, o sistema exemplifica a separação clara entre a lógica de negócios e as tecnologias externas.

Características principais:

  • Domínio de negócios isolado e independente de frameworks
  • Adaptadores flexíveis para UI, banco de dados e serviços externos
  • Fácil testabilidade e manutenção
  • Autenticação e autorização com Tokens JWT

Explore o código fonte no GitHub para ver como a arquitetura hexagonal melhora a estrutura e a escalabilidade do projeto.
